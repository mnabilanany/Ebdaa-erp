# 🏢 Ibdaa Real Estate ERP - نظام ERP إبداع العقاري

**النسخة**: 2.0 (محسّن بالأمان والأداء)

## 📋 نظرة عامة

نظام إدارة مؤسسات (ERP) متكامل لشركات التطوير العقاري، يجمع بين:
- ✅ إدارة المشاريع والعقارات
- ✅ العقود والأقساط
- ✅ المحاسبة والقوائم المالية
- ✅ إدارة الموارد البشرية
- ✅ التحليلات والتقارير

---

### 🏗️ إدارة العقارات
- تسجيل المشاريع والوحدات

### 💰 إدارة الأقساط
- توليد الأقساط الشهرية
- تتبع السداد والتأخر
- تنبيهات الأقساط المتأخرة

### 📊 المحاسبة والتقارير
- قيود يومية ودفتر أستاذ
- ميزان المراجعة
- قائمة الدخل والميزانية العمومية
- تقارير تفصيلية بصيغة PDF

### 👨‍💼 إدارة الموارد البشرية
- إدارة الموظفين والأقسام
- الرواتب والمزايا
- سجلات الموظفين

## 🚀 البدء السريع

### المتطلبات:
```bash
Python 3.8+
pip
SQLite3
```

### التثبيت:
```bash
# 1. نسخ المستودع أو فك الضغط
cd ibdaa_erp

# 2. تثبيت المكتبات
pip install -r requirements.txt

# 3. الإعدادات
cp .env.example .env

# 4. تشغيل النظام
python app.py
```

### الوصول:
- **URL**: `http://localhost:5000`
- **المستخدم**: `admin`
- **كلمة المرور**: `1234`

⚠️ **تغيير كلمة المرور فوراً!**

## 🔄 تحديثات ونسخ احتياطية

### إنشاء نسخة احتياطية:
```python
from backup_manager import BackupManager

mgr = BackupManager("erp.db", "backups/")
mgr.create_backup()
mgr.cleanup_old_backups(30)  # حذف أقدم من 30 يوم
```
