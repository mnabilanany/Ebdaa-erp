# 🏗 دليل تثبيت نظام إبداع ERP
## نظام ERP العقاري الويب — شركة إبداع

---

## 📦 المتطلبات

- Python 3.10 أو أحدث
- اتصال بالإنترنت (للتثبيت الأول فقط)
- نظام تشغيل: Windows / Linux / macOS

---

## 🖥 التثبيت على الجهاز المحلي (شبكة داخلية)

### الخطوة 1 — تثبيت Python
من الموقع الرسمي: https://python.org/downloads
✅ تأكد من تفعيل "Add Python to PATH" أثناء التثبيت

### الخطوة 2 — ضع ملفات المشروع
افتح مجلد المشروع:
```
ibdaa_erp/
├── app.py
├── requirements.txt
├── templates/
│   ├── base.html
│   ├── login.html
│   ├── dashboard.html
│   ├── journal.html
│   ├── journal_detail.html
│   ├── ledger.html
│   ├── reports.html
│   ├── properties.html
│   ├── clients.html
│   ├── contracts.html
│   ├── installments.html
│   ├── hr.html
│   ├── coa.html
│   ├── engineering.html
│   ├── analytics.html
│   ├── users.html
│   └── settings.html
├── static/
│   ├── css/style.css
│   └── js/main.js
```

### الخطوة 3 — تثبيت المكتبات
افتح CMD أو Terminal داخل مجلد المشروع:
```bash
pip install -r requirements.txt
```

### الخطوة 4 — تشغيل النظام
```bash
python app.py
```

ثم افتح المتصفح على:
```
http://localhost:5000
```

---

## 🌐 تشغيل على شبكة داخلية (LAN) — كذا مستخدم في نفس الوقت

### على الجهاز الخادم (Server PC):
```bash
python app.py
```
البرنامج يشتغل على `0.0.0.0:5000` تلقائياً

### على أجهزة المستخدمين الأخرى:
افتح المتصفح واكتب:
```
http://[IP-الجهاز-الخادم]:5000
```

مثال: لو IP الخادم هو `192.168.1.10`:
```
http://192.168.1.10:5000
```

### معرفة IP الجهاز الخادم:
- Windows: افتح CMD واكتب `ipconfig`
- Linux/Mac: افتح Terminal واكتب `ip addr` أو `ifconfig`

## ☁ النشر على الإنترنت (Hosting) — للوصول من أي مكان

### الخيار 1: Railway.app (مجاني / سهل)
1. اذهب لـ https://railway.app
2. سجل حساب بـ GitHub
3. اضغط "New Project" → "Deploy from GitHub repo"
4. ارفع المشروع على GitHub أولاً

### الخيار 2: Render.com (مجاني)
1. اذهب لـ https://render.com
2. أنشئ "New Web Service"
3. ارفع المشروع
4. ضع في "Start Command":
```
gunicorn app:app
```
5. أضف في requirements.txt:
```
gunicorn
```

### الخيار 3: VPS (DigitalOcean / Linode / Hetzner)
```bash
# على الـ VPS
sudo apt update && sudo apt install python3 python3-pip nginx -y
pip3 install gunicorn flask openpyxl
cd /var/www/ibdaa_erp
gunicorn -w 4 -b 0.0.0.0:8000 app:app
```

## 🔧 استكشاف الأخطاء

| المشكلة | الحل |
|---------|------|
| `ModuleNotFoundError: flask` | شغّل `pip install flask openpyxl` |
| الصفحة لا تفتح | تأكد أن البرنامج شغّال وافتح `localhost:5000` |
| مستخدمون آخرون لا يصلون | تأكد أنهم على نفس الشبكة وابحث عن IP الخادم |
| نسيت كلمة المرور | احذف ملف `erp.db` وأعد التشغيل (بيتعمل admin/1234 جديد) |
